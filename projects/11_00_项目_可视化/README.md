# 棒球数据可视化

## 构思
1. 导入数据，用dimple.js画气泡图，身高体重分别为横竖坐标，气泡大小颜色固定，注意数据缩放；
2. 气泡大小代表全垒得分，气泡颜色深浅代表命中率高低；
3. 加入用手习惯选择
